
# Object Detection Using TensorFlow

## Overview

This project demonstrates object detection using TensorFlow and computer vision techniques. The notebook detects and visualizes landmarks from input images using a deep learning model.

## Features

- Object detection using TensorFlow
- Image preprocessing
- Landmark visualization
- Easy-to-understand Jupyter Notebook implementation
- Well-structured workflow from input image to output

## Technologies Used

- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook

## Installation

1. Clone the repository:

```bash
git clone https://github.com/karankirtan-a11y/Object-Detection-Using-TensorFlow.git
````

2. Navigate to the project folder:

```bash
cd Object-Detection-Using-TensorFlow
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Open the notebook:

```bash
jupyter notebook
```

## How It Works

1. Load the input image.
2. Preprocess the image.
3. Run the TensorFlow object detection model.
4. Extract landmark coordinates.
5. Display the detected objects on the image.

## Results

The notebook successfully identifies and visualizes objects on the input images. The outputs demonstrate TensorFlow's capability to perform accurate object detection.
